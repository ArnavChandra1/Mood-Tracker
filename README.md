# Mood Tracker (Android)

A lightweight Android application for logging and tracking daily emotional states.

## Overview

Mood Tracker allows users to record emotions and view a history of logged entries. The app focuses on simplicity, clean UI, and persistent local storage.

## Features

- Log emotional states 
- View list of previously recorded entries
- Navigate between multiple activities
- Persistent local storage
- Clean and responsive UI
---

## Tech Stack

- Java
- Android SDK
- Android Studio
- Gradle (Kotlin DSL)
- XML-based layouts
- Local file storage

---

## How to Run

1. Clone the repository:
   ```bash
   git clone git@github-arnav:ArnavChandra1/Mood-Tracker.git
   ```
2. Open the project in Android Studio  
3. Allow Gradle to sync  
4. Run on an emulator or Android device
